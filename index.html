¿Qué es VueJS?

Vue ya tiene una dilatada historia. En 2014 Evan You acababa de terminar su etapa en Google como desarrollador y entre otros proyectos también había trabajado en Angular. Su experiencia le permitió crear su propia biblioteca personal que posteriormente liberó al público. Fue precisamente la comunidad quien hizo crecer el proyecto hasta convertirse en el potente y ligero framework que es en nuestros días.

Framework progresivo Javascript para crear interfaces de usuario. Está diseñado desde cero para ser utilizado incrementalmente.

Características

MODULARIDAD

en Vue tienes más libertad para usar ciertas características como por ejemplo: realizar llamadas HTTP hace falta la utilización de un paquete como axios para hacer uso de dicha característica. Todo esto nos da a conocer que vue te ofrece lo básico para trabajar, pero tú decides si instalar a posteriori utilidades que te serán de ayuda en ciertos aspectos de tu página web.

REACTIVIDAD

Vue contiene propiedades reactivas, eso quiere decir que si se cambia una variable en cierta página, este framework actualiza su valor sin necesidad de hacerlo manualmente. Un ejemplo claro es el uso de un contador.

COMPONENTES WEB

Vue se basa en componentes web, este se puede definir como una parte de una web que puede ser reutilizada y que normalmente tiene estilos y funcionalidades. Los componentes web pueden ser creados con JavaScript.

VIRTUAL DOM

esto tiene un valor importante al momento de realizar cambios en la vista, haciendo que en lugar de sustituir directamente valores en la vista Vue crea una réplica del Dom que facilita el o los cambios de forma más óptima.

EVENTOS Y TRANSICIONES

estos reaccionan a los eventos provenientes en el DOM y es más usado cuando el usuario interactúa con elementos dentro de la página web. Su sistema de transiciones y animaciones es sencillo de usar, pero a su vez puede llegar a ser muy complejo.

MIXINS

son funciones y lógica de los componentes que se puede reutilizar en otros componentes web.

CICLO DE VIDA DE LOS COMPONENTES

se tiene el control sobre todo el ciclo de vida de los componentes, es decir, se puede controlar lo que ocurre antes de que se cargue el componente. Esto es importante o útil por ejemplo cuando queremos llamar a un API REST para la carga de datos.

APLICACIONES SPA

Vue ofrece un sistema de rutas llamado VUE ROUTER que permite tener una página SPA que conlleva a que todas las páginas se encuentran ya cargadas cuando el usuario entra en la página web de tal forma que cambiar de página es instantáneo.

    Ventajas de VueJS

    En cuanto a su curva de aprendizaje se tarda menos tiempo en aprender lo básico para poder desenvolverse en este framework.

    Posee una arquitectura limpia, el código de los componentes queda mucho más ordenado y limpio.

    Los archivos dentro de Vue permiten la escritura de HTML y CSS sin tener que hacerlo dentro de JavaScript.

    Fuera de un aspecto de programación, Vue contiene una comunidad que continuamente se encuentra en crecimiento, tal vez no tanto como React o Angular, pero sin duda se perfila para ser uno de los framework más utilizados en los siguientes años.

    Instalación

    # Paso de Instalación Comando
    1 La instalación de Vue es bastante sencilla, basta hacer uso de Vue CLI la cual es una herramienta para línea de comandos que sirve para gestionar proyectos de Vue, para utilizarla podemos descargarla en el equipo con el uso de NPM: npm install -g @vue/cli
    2 Seguidamente ya instalado podemos crear la estructura inicial de nuestro proyecto con el comando: vue create nombre_del_proyecto
    3 Una vez creado el proyecto podremos crear nuestro proyecto eligiendo las configuraciones del mismo. Podemos elegirlo por defecto que es lo más común, pero también podemos personalizarlo según el alcance de nuestro proyecto. Para la ejecución de nuestro proyecto bastará con insertar el comando: npm run serve
    4 Si se desea ejecutar nuestro proyecto mediante una interfaz gráfica que lo que nos permite es administrar los proyectos de Vue, basta con insertar la siguiente línea de comando: vue ui
